Using the Spring Boot starters for Azure, developers can easily integrate Azure cloud-based technologies into their applications. In addition, the Java SDK for Azure allows developers to leverage additional Azure features. {% endcapture %} {% capture main_content %} ## Features - [Spring Data Azure DocumentDB](https://github.com/Microsoft/spring-data-documentdb): Spring Data module for Microsoft Azure DocumentDB.

- [Spring Security Azure AD](https://azure.microsoft.com/en-us/blog/spring-security-azure-ad/): Enterprise-grade authentication and authorization for Spring Security.

- Spring Boot Starters provide auto-configuration support for the following Azure services: - **[Azure Active Directory](https://docs.microsoft.com/azure/active-directory/)**: Provides Azure integration with Spring Security for authentication and authorization. - **[Azure Key Vault](https://azure.microsoft.com/services/key-vault/)**: Manage application secrets and keys. - **[Azure Storage](https://docs.microsoft.com/azure/storage/)**: Support for cloud-based blob storage. - **Additional Azure Features**: Auto-configuration support for additional Azure features like [Azure Service Bus](https://docs.microsoft.com/azure/service-bus/), [Cosmos DB](https://docs.microsoft.com/azure/cosmos-db/), etc.

- Azure provides Spring-based support for additional cloud-computing features: - **Containers**: For an example, see [Deploy a containerized Spring Boot app to Azure](https://docs.microsoft.com/java/azure/spring-framework/deploy-containerized-spring-boot-java-app-with-maven-plugin) - **Kubernetes** and **Docker**: For an example, see [Deploy a Spring Boot Application on a Kubernetes Cluster in the Azure Container Service](https://docs.microsoft.com/java/azure/spring-framework/deploy-spring-boot-java-app-on-kubernetes) - **Cosmos DB**: For an example, see [How to use the Spring Boot Starter with Azure Cosmos DB DocumentDB API](https://docs.microsoft.com/java/azure/spring-framework/configure-spring-boot-starter-java-app-with-cosmos-db)

- Azure provides support for the following Spring Data projects through the Azure data services that are listed below: - **Spring Data JPA**: [Azure MySQL](https://azure.microsoft.com/services/mysql), [Microsoft SQL Server](https://docs.microsoft.com/azure/sql-database/) - **Spring Data JDBC**: [Azure MySQL](https://azure.microsoft.com/services/mysql) - **Spring Data Mongodb**: Azure MongoDB (via [Azure CosmosDB](https://azure.microsoft.com/services/cosmos-db/)) - **Spring Data MySQL**: [Azure Database for MySQL](https://azure.microsoft.com/services/mysql/) (For an example, see [Build a Java and MySQL web app in Azure](https://docs.microsoft.com/azure/app-service/app-service-web-tutorial-java-mysql).) - **Spring Data for Apache Cassandra**: Azure Cassandra (via [Azure CosmosDB](https://azure.microsoft.com/services/cosmos-db/)) - **Spring Data Redis**: [Azure Redis Cache](https://azure.microsoft.com/services/cache/) (For an example, see [Configure a Spring Boot Initializer app to use Redis in the cloud with Azure Redis Cache](https://docs.microsoft.com/java/azure/spring-framework/configure-spring-boot-initializer-java-app-with-redis-cache).) ## Quick Start {% include download_widget.md %} The following excerpt illustrates how to integrate the Spring starters for Azure into your `pom.xml` file: ```xml com.microsoft.azure azure-spring-boot com.microsoft.azure azure-active-directory-spring-boot-starter com.microsoft.azure azure-keyvault-secrets-spring-boot-starter com.microsoft.azure azure-storage-spring-boot-starter com.microsoft.azure azure-spring-boot-bom ${azure.version} pom import ``` {% endcapture %} {% capture related_resources %} ### Getting Started Guides The following articles will show you how to get started with Spring Cloud for Azure: -
